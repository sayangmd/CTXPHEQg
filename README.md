# 项目简介

## 核心功能
- 用户管理
- 个人中心
- 评论管理
- 订单管理
- 商品信息管理
- 团购信息管理
- 注册登录
- 查看商品
- 首页
- 购物车
- 我的订单
- 团购信息
- 评论留言
- 网站资讯

## 适用场景
- 酒店管理
- 社区团购
- 后台管理

## 技术特点
- 基于Spring Boot框架
- 使用Vue脚手架搭建后台管理界面
- 支持多种IDE开发，如Idea、Eclipse、Vscode
- 数据库支持MySQL
- 构建工具Maven
- 环境要求：JDK1.8、Node16、Mysql8、Maven3.6
